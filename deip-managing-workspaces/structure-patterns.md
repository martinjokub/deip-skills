## Naming Conventions

- Use clear, descriptive names
- Avoid abbreviations unless universally understood
- Be consistent across the workspace
- Consider how items will be searched/filtered

## Tips

1. **Start broad**: Create 3-5 top-level categories first
2. **Add subcategories as needed**: Don't over-structure upfront
3. **Review regularly**: Adjust structure as ecosystem evolves
4. **Use notes for context**: Add descriptions to categories when helpful
